# 写在前面
基于NodeJS 环境，使用Express + Mongoose + Bootstrap + EJS 从零开发的一个管理系统和博客主页

#### 接着
很多人会问，为什么要用EJS来做，这不是老古董了吗？ 现在大家都用React Vue. 正是因为在这个前端百花齐放的年代，很少有一个从前的传统模板渲染的项目，我选择这种朴实的方式来做一个个人的小项目。
